# juju
A multipurpose bot that runs on Discord.

# Running
1. Make sure to get Node.js 16.16.0 or higher and TypeScript 5.1.3 or higher
2. Run install.bat
3. Setup configuration (src/config.json)
4. Run start.bat

# Commands
1. /balance <user>
2. /work
3. /daily
4. /gamble <amount>
5. /leaderboard
6. /help
7. /setlogchannel <channel>
8. /removelogchannel
9. /source
10. /clean <amount>
